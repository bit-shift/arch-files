Install the tmux package 
	- pacman -S tmux

Create the .tmux.conf file if it does not already exist
	- touch ~/.tmux.conf

Install the tmux plugin manager 
	- git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
	- Put the following lines into the ~/.tmux.conf file
	- set -g @plugin 'tmux-plugins/tpm'
	- run -b ~/.tmux/plugins/tpm/tpm
	- The above line must be the last line in the file
	- Add other plugins in the same format below the first plugin
	- set -g @plugin 'plugin/name'

Using the plugin manager
	- Add the new plugin using the syntax above, and write the file
	- Open tmux, and press prefix(C-a)+I
	- prefix(C-a)+U updates plugins
	- To remove a plugin, comment out the plugin, and write the file
	- Open tmux, and press prefix(C-a)+alt+U
