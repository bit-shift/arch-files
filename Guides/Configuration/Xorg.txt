Download the xorg-server, xorg-xinit, and xorg-xrdb packages
	- pacman -S xorg-server xorg-xinit xorg-xrdb

Identify the video card and dowload the appropriate driver
	- lspci | grep -e VGA -e 3D
	- Search for the xf86 video drivers 
	- pacman -Ss xf86-video
	- Select the xf86-video-intel driver
	- pacman -S xf86-video-intel

Create the .xinitrc file 
	- vim ~/.xinitrc
	- Might need to make it a bash script, add the shebang and make executable
	- Add the following lines to the .xinitrc
	- [[ -f ~/.Xresources ]] && xrdb -merge ~/.Xresources
	- exec i3 
	- xrdb updates the settings based on the .Xresources file

Create the .Xresources file
	- vim ~/.Xresources
	- See file for configurations of urxvt
	- Use urxvt --help for all config options
	- * is essentially a wildcard matcher
	- Can use the URxvt. syntax instead, but not as wide coverage
	- Used terminal.sexy to get the colourscheme
	- Colourscheme used was baskerville - FarSide

Create the symlink for .Xdefaults
	- ln -sf .Xresources .Xdefaults
	- In case files look to .Xdefaults for loading custom values
	- Old software uses .Xdefaults, but good to be safe

Refer to the i3.txt for installation of xorg-xdpyinfo and xorg-xrandr
