Download the required packages for i3-gaps
	- pacman -S i3-gaps i3status

Download the terminal emulator urxvt
	- pacman -S rxvt-unicode
	- See Xorg.txt for info on customisation

Run the X server for the first time executing i3
	- Use the files from Bash.txt and Xorg.txt
	- When run for the first time, get i3 to generate a default config file
	- Located at ~/.config/i3/config
	- See file for changes and improvements made to the file
	- Based off of LukeSmithxyz git repo

Install wget and feh packages for wallpaper
	- pacman -S wget feh
	- wget <url>
	- Move the file to the desired location
	- feh is used to load the wallpaper

Install the i3lock and imagemagick packages
	- pacman -S i3lock imagemagick
	- Add the key binding to the .config/i3/config file
	- Convert the image to .png format
	- convert -scale 2560x1440 -blur 0x8 source-image.jpg dest-image.png
	- i3lock -e -f -i /path/to/image.png

Install the betterlockscreen package
	- Install the required packages xorg-xdpyinfo, xorg-xrandr, imagemagick, and bc
	- pacman -S xorg-xdpyinfo xorg-xrandr bc
	- Install the i3lock-color AUR
	- pakku -S i3lock-color
	- This requires i3lock to be removed 

Clone into the git repository
	- git clone https://github.com/pavanjadhaw/betterlockscreen ~/Git/betterlockscreen
	- cd ~/Git/betterlockscreen
	- mkdir ~/.local/bin
	- cp betterlockscreen ~/.local/bin/
	- Update the PATH variable in .profile
	- export PATH="$PATH:$HOME:$HOME/.local/bin/"

Add the following lines to activate lockscreen
	- bindsym $mod+Shift+x				exec --no-startup-id betterlockscreen -l -t "keegan@rogue"
	- bindsym $mod+Shift+backslash		exec --no-startup-id betterlockscreen -l -t "keegan@rogue" & sleep 0.1 && $hibernate

Use the i3blocks statusbar
	- bar { status_command i3blocks }
	- Refer to the config file for the details of the bar config

Reset the key bindings for gap resizing
	- Refer to the config file for the exact changes
	- Issues with the outer gaps piling on top of the inner gaps
	- Refer to the Issues/ directory for the proper write up
	- Solved

Install the xorg-xev package
	- pacman -S xorg-xev
	- xev -event keyboard
	- Use this command to get the name of the key pressed 
	- Helpful for key bindings to non-standard keys
