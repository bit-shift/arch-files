Install the libnotify and dunst packages
	- pacman -S libnotify dunst

Make the dunst directory in the ~/.config directory 
	- mkdir ~/.config/dunst

Create the dunstrc file in the ~/.config/dunst/dunstrc
	- v ~/.config/dunst/dunstrc
	- Copy across the default file
	- cp /usr/share/dunst/dunstrc ~/.config/dunst/dunstrc

Start dunst on execution of i3
	- exec_always command re-executes dunst after reload
	- Bind a key to kill dunst in order to reload config file
	- Refer to the i3 config file for bindings
