Vim is already installed during installation process per notes
	- pacman -S gvim

Create the .vimrc file
	- vim ~/.vimrc
	- Refer to the file for the specific changes made
	- Influenced by Luke Smith's .vimrc

Set up Vundle as the Plugin manager
	- set nocompatible
	- filetype off
	- set rtp+=~/.vim/bundle/Vundle.vim
	- call vundle#begin()
	- Plugin 'VundleVim/Vundle.vim'
	- call vundle#end()
	- filetype plugin indent on
	- Run the following commands to install the Plugin
	- vim
	- :PluginInstall

Installing Plugins 
	- Plugin 'Plugin-name'
	- vim
	- :PluginInstall
	- Vundle Plugin must be the first Plugin

Install the onedark.vim colorscheme
	- Plugin 'joshdick/onedark.vim'
	- Use the Plugin installation process above
	- colorscheme onedark

Install the lightline status bar
	- Plugin 'itchyny/lightline.vim'
	- Use the Plugin installation process above
	- set laststatus=2
	- To make the colorscheme match the onedark scheme
	- let g:lightline = {
	-	\ 'colorscheme': 'onedark',
	-	\ }

Install the youcompleteme plugin
	- Plugin 'Valloric/YouCompleteMe'
	- Use the Plugin installation process above

Install the clang and cmake packages
	- pacman -S clang cmake 
	- Will be used in the installation process for the c-family languages

Install youcompleteme
	- cd ~/.vim/bundle/YouCompleteMe
	- python3 install.py --clang-completer
	- Create the .ycm_extra_conf.py file 
	- vim ~/.ycm_extra_conf.py

Configure the .ycm_extra_conf.py file
	- Refer to the .ycm_extra_conf.py file for the basics
	- The flags returned in the Settings function are the flags used on compilation

Configure the .vimrc file 
	- Set the ycm_global_ycm_extra_conf variable for a fallback
	- let g:ycm_global_ycm_extra_conf = '/home/keegan/.ycm_extra_conf.py'
	- Set the ycm_confirm_extra_conf variable to disable confirmation
	- let g:ycm_confirm_extra_conf = 0

Install vimtex
	- Plugin 'lervag/vimtex'
	- Use the Plugin installation process above

Configure the .vimrc file
	- Ensures tex files are properly recognised, and zathura is the pdf viewer
	- let g:tex_flavor = 'latex'
	- let g:vimtex_view_method = 'zathura'
	- Ensure youcompleteme works for LaTeX
	- Refer to the .vimrc file

Write the vimura script, located in the ~/.vim/ directory
	- Refer to the Vim + Zathura + Synctex github
	- Refer to the .vimrc file for the reverse search
	- Load with vimura command to enable reverse search
	- Script is buggy and causes issues in terminal
	- Need to close .tex file before opening .pdf file
