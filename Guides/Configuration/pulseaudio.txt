Install the pulseaudio and pulseaudio-alsa packages
	- pacman -S pulseaudio pulseaudio-alsa 
	- pulseaudio-alsa is used to work with the alsa config

Run pulseaudio as a daemon
	- pulseaudio --daemonize
	
Install the pamixer package 
	- pacman -S pamixer
	- Allows for easy control of volume and mute from the command line
	- Refer to the i3 config file for the commands and bindings

Install the pulseaudio-bluetooth package
	- pacman -S pulseaudio-bluetooth
	- Used for bluetooth interface between bluez and pulseaudio
