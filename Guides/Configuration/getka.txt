Write the getkeys and getaliases scripts
	- Place them in the ~/.scripts/tools directory
	- Refer to the files for the scripts themselves
	- Add the ~/.scripts/tools directory to the path

Write the getkeys and getaliases files
	- Place them in the ~/.config/getkeys and ~/.config/getaliases directories
	- Refer to the files inside the directories for the contents
	- Provide the key bindings and aliases for the particular program or alias group

Run the scripts as just the command gives the list of arguments
Run the scripts with an argument gives the key bindings or aliases for the argument
